# Report a Bug

Bugs can be reported [here](https://bugs.launchpad.net/snappy-hwe-snaps/+filebug).

When submitting a bug report, please attach:

 * */var/log/syslog*

And the output of the following commands:

```
$ sudo pactl list
$ cat /proc/asound/cards
```
